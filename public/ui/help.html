Quick Troubleshooting

  • If you cannot buy a ticket, ensure the server is running and the DB is initialized.
  • Use the Idempotency-Key header when creating orders to prevent duplicates.
  • Cashiers can scan QR or enter order id on the cash desk screen.
Need more help? Open an issue in the repo or contact the dev team.
